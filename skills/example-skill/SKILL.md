# Example Skill

## When to Use This Skill

Use this skill as a template when creating new skills. This demonstrates:
- Proper YAML frontmatter format
- Section structure
- Documentation style

## The Process

**Step 1: Understand the trigger**

Identify when this skill should activate. Be specific about the conditions.

**Step 2: Execute the workflow**

Document the exact steps to follow. Use:
- Clear headings
- Numbered/bulleted lists
- Code blocks where appropriate
- Examples

**Step 3: Verify completion**

Describe how to know the skill has been successfully applied.

## Key Principles

- Be explicit about when to use this skill
- Provide step-by-step instructions
- Include examples
- Make it easy to follow
